انتقل إلى المحتوى الرئيسي وَرَتِّلِ الْقُرْآنَ تَرْتِيلًا منصة القرآن والحديث واللغة العربية رحلة تعليمية هادئة وتفاعلية للحفظ والفهم والتعلّم لغة المنصة العربية سيتم حفظ اختيارك وتطبيقه على صفحات المنصة. ابدأ رحلتك التعليمية اختر القسم الذي تريد دراسته ثلاثة أقسام مترابطة تساعدك على الحفظ والمراجعة والفهم خطوة بخطوة. 📖 حفظ القرآن الكريم استمع إلى التلاوة، أخفِ الكلمات أو الآيات، وتابع القراءة بطريقة الكاراوكي. اختيار القارئ المفضّل تمارين إخفاء الكلمات والآيات تظليل متزامن مع الصوت دخول قسم القرآن 🕌 الحديث النبوي الشريف اقرأ الأحاديث واحفظها مع شرح موجز وترجمات سهلة الإضافة والتحديث. نص الحديث مضبوط وواضح ترجمات باللغات الست إضافة المحتوى من ملف JSON بسيط دخول قسم الحديث ✍️ تعلّم اللغة العربية دروس مبسطة في الحروف والمفردات والقواعد لفهم العربية تدريجياً. دروس قصيرة ومنظمة أمثلة وتمارين تطبيقية إضافة الدروس من ملف JSON بسيط دخول قسم اللغة العربية 💡 واجهة بسيطة وقابلة للتوسّع ستقرأ المنصة الترجمات والأحاديث والدروس من ملفات JSON منفصلة، لذلك يمكنك تحديث المحتوى من GitHub دون تعديل الكود البرمجي. © منصة القرآن والحديث واللغة العربية
